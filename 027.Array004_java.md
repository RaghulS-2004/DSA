
## Problem-26:
Smallest Number in Array:

## PROGRAM:(Main.java)
```
import java.util.*;

class Main {
    public static void main(String[] args) {
        Scanner input=new Scanner(System.in);
        int n = input.nextInt();
        int [] arr = new int [n];
        
        for(int i=0 ;i<n ;i++)
        {
            arr[i]=input.nextInt();
        }
        int small=arr[0];
        for(int i=1 ;i<n ;i++)
        {
            if(arr[i]<small)
            {
                small=arr[i];
            }
        }
        System.out.println("Smallest number : "+small);
    }
}
```
