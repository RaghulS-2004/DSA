
## Problem-1:
Write a python program for a palindrome of a number
Input: 12345

Output:
```
54321
```

## PROGRAM:(Main.java)
```
import java.util.*;
public class Main
{
    public static void main(String[] args)
    {
        Scanner input= new Scanner(System.in);
        int n = input.nextInt();
        int sum=0;
        while(n>0)
        {
            int ld = n%10;
            sum=sum*10+ld;
            n/=10;
        }
        System.out.print(sum);
    }
}
```
